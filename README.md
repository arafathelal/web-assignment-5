# web-assignment-5

# var is function scoped, allows redeclaration and reassignment. let is block scoped, allows reassignment but not redeclaration. const is is block scoped, does not allow reassignment.

# spread operator (...) is used to copy, merge arrays in a simple way. it allows to expand items from an array or object to individal items

# map() creates a new array by modifying each element. filter() returns a new array with elements that match a condition. forEach() runs a function for each element and does not return a new array.

# arrow function is a shorter way to write functions using the arrow syntax =>, it was introduced in ES6

# template literal is a way to create strings in js using backticks ``. it allows using variables and expression inside string using ${}.

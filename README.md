# 🚀 Git Issue Tracker

A simple web-based GitHub Issue Tracker that allows users to log in, view issues, search issues, and inspect detailed issue information in a clean and responsive interface.

---

## 📌 Features

* 🔐 Basic login system (frontend-based)
* 📋 View list of issues
* 🔍 Search issues by keyword
* 📄 Detailed issue view (modal)
* 📱 Fully responsive UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Tailwind CSS
* DaisyUI
* Font Awesome

---

## 📁 Project Structure

```
Git-Issue-Tracker/
│── assets/              # Images, icons, etc.
│── index.html           # Login page
│── issue.html           # Issue dashboard
│── login.js             # Handles login logic
│── issue.js             # Fetch & display issues
│── tailwind.init.css    # Tailwind setup
```

---

## ⚙️ How It Works

1. User lands on the **login page** (`index.html`)
2. After login, user is redirected to **issue dashboard** (`issue.html`)
3. Issues are fetched from an external API
4. Users can:

   * Search issues 🔍
   * Click to view full details 📄

### 3️⃣ Open in Browser

You can run this project in multiple ways:

#### ✅ Option 1: Direct Open

* Open `index.html` in any browser

#### ✅ Option 2: Live Server (Recommended)

1. Open project in VS Code
2. Install **Live Server extension**
3. Right-click `index.html`
4. Click **"Open with Live Server"**
